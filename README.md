# Viveros El Ciprés Web

Responsive landing page developed for **Viveros El Ciprés**, a plant nursery and gardening business located in Beniaján, Murcia.

The website presents the business, its main product areas, services, contact information and direct conversion links such as WhatsApp, Google Maps and Instagram.

This project is built with **HTML5** and **CSS3**, without external frameworks, and is prepared for deployment with **GitHub Pages** using a custom domain.

## Project Overview

This project is a simple and lightweight business website focused on clarity, usability and local conversion.

The website includes:

- Business presentation.
- Product and service sections.
- Contact information.
- WhatsApp call-to-action.
- Google Maps integration.
- Instagram link.
- Responsive layout for mobile and desktop.
- Custom domain support through `CNAME`.

---

## Website Sections

### Header

Sticky navigation bar with internal links to the main sections of the page.

### Hero Section

Main presentation block with a short value proposition and two conversion buttons:

- WhatsApp contact.
- Google Maps directions.

### Products

Overview of the main product categories:

- Plants and nursery products.
- Agriculture and irrigation.
- Growshop products.
- Gardening and agricultural tools.

### Services

Summary of the main services and customer value:

- Personalised customer service.
- Landscaping guidance.
- Solutions for professionals and private customers.

### Trust Section

Highlights key selling points such as selected materials, direct attention, experience and professional advice.

### Contact

Includes address, email, WhatsApp, Instagram and embedded Google Maps location.

## Local Usage

Clone the repository:

```bash
git clone https://github.com/miguelangeldealba/viveros-el-cipres-web.git
cd viveros-el-cipres-web
```

Open the website locally by opening `index.html` in a browser.

No installation is required.

---

## Deployment

This project is prepared for deployment through GitHub Pages.

The `CNAME` file points to the custom domain:

```text
viveroselcipres.es
```

Recommended GitHub Pages configuration:

```text
Branch: main
Folder: /root
```
